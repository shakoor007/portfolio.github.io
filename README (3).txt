Asset List
The asset list details all the assets included in the project folder:
* index.html - The homepage for the portfolio.
* about.html - The page detailing information about Anas Shakoor.
* contact.html - The contact page with a form for users to submit inquiries.
* styles.css - The stylesheet that contains all the styling for the website.
* script.js - The JavaScript file that contains the logic for interactive features such as notifications, theme changing, and form validation.
* IMG_1194.jpg - An image file used in the index.html for profile display.
* unnamed.png - An image file depicting the workplace, used in about.html.
* sabeel.png - An image file depicting the community center, used in about.html.
* favicon.ico - The favicon icon file for the website.
